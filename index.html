Getting Started

Up and Running
in 5 Minutes.

Download, install inside your existing software, and start designing with AI. No new workflow to learn.

01
Download CarvAI
Download the plugin file from carvai.com. Works as a VCarve/Aspire gadget or standalone PWA on any device.
02
Install in Vectric
VCarve/Aspire: Gadgets → Install Gadget → select the .vce file. The CarvAI toolbar appears instantly.
03
Open CarvAI
Click the CarvAI button in your toolbar. The AI panel opens alongside your existing design project.
04
Design with AI
Say or type what you want. "¾ inch roundover all edges." CarvAI applies it live with the exact bit spec.
05
Export & Cut
One click exports to your machine — X-Carve, xTool, VCarve, or any other. No format conversion needed.
